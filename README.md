A guide to making tinctures, written for the purpose of being migrated to Azure cloud as capstone project for General Assembly Software Engineering Bootcamp.

## <a name="userstories"></a> User Stories

* AAU, I want the aesthetic of the app to be clean and reminiscent of a notebook of home remedies.
* AAU, I want to be able to view a page containing links to all of my tinctures as well as a link to create a new recipe.
* AAU, I want to be able to add a new tincture/tonic recipe.
* AAU, I want to be able to edit or delete existing tincture/tonic recipes.

## <a name="features"></a>Features
* User can view details of individual herbal ingredients or tinctures
* User can edit existing details for individual herbal ingredients or tinctures
* User can add a new herbal ingredient or tincture
* User can delete an existing herbal ingredient or tincture
